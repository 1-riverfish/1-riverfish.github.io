Linux入门基础3

Linux文件基本操作管理

cp(copy) 复制文件或目录 cp 源文件(文件夹) 目标文件(文件夹)

常用参数 -r 递归复制整个目录树 -v 显示详细信息 -r参数和目录有关

移动，重命名文件或目录

通过mv命令移动或者重命名文件或目录

mv 文件 目标目录

mv 文件 目标目录/文件名

如果指定文件名，则可以重命名文件

mv 文件名 新文件名

rm(remove)

参数:

-r 递归删除文件夹

-i 交互式删除

-f 强制删除，没有警告提示(使用时需谨慎)

创建，删除目录

mkdir 创建一个目录

rmkdir删除一个空目录

rm -r (-f)命令删除一个非空目录
