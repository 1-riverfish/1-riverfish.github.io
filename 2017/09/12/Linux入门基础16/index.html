Linux入门基础16

Linux网络基础配置

以太网连接

  • 在Linux中，以太网接口被命名为eth0,eth1 ,0,1代表网卡编号
  • lspci 查看网卡硬件信息（lsusb 查看usb网卡）(pci接口)
  • ifconfig 查看接口信息
    • -a 查看所有接口
    • eth0 查看特定接口
  • ifup ifdown 开启 禁用接口

setup 命令

DHCP动态分配IP地址

网络相关配置文件

  • 网卡配置文件

/etc/sysconfig/network-scripts

  • DNS配置文件

    /etc/resolv.conf

  • 主机名配置文件

    etc/sysconfig/network

网络测试命令

  • 测试网络连通性

ping 192.168.1.1

ping www.baidu.com

  • 测试DNS解析

host www.baidu.com

dig www.baidu.com

  • 显示路由表

ip route

  • 追踪到达目标地址的网络路径

traceroute www.baidu.com

  • 使用mtr进行网络质量检测（结合了traceroute和ping）(My Traceroute)

mtr www.baidu.com

修改当前主机名

实时修改hostname newname

永久修改/etc/sysconfig/network

故障排查

从底层到高层，从自身到外部

  • 网络配置是否正确
    • IP 子网掩码 DNS
  • 查看网关是否连通
    • ping 网关IP地址
  • 查看DNS解析是否正常
    • mtr traceroute
