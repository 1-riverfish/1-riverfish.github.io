Linux入门基础5

Linux系统常用命令

日期时间

date 查看，设置当前系统的时间，+%--%m--%d格式化显示时间 date -s "XX:XX:XX"修改时间

hwclock(clock) 用以显示硬件时钟时间

cal 用以查看日历

uptime 查看系统运行时间,启动时间

输出，查看命令

echo 用以显示输入的内容

cat 显示文件内容

head 显示文件的头几行 -n nub 默认显示10行

tail ~的末尾几行 -n -f (follow) 追踪显示文件更新（一般用于查看日志，命令不会退出，而是持续显示新加入的内容）

more 翻页显示文件内容(空格键下翻) q

less 翻页显示文件内容(page up/dowm上下翻) q退出

查看硬件信息

lspci 查看PCI设备 -v 查看详细信息(硬件设备)

lsusb 查看USB设备 -v 查看详细信息

lsmod 查看加载的模块（驱动）

关机，重启

shut down -h(hold)关机 -r(reboot)重启

shut down -h now / +10 /23:30

poweroff 立即关闭计算机

reboot 立即重启计算机

归档，压缩

zip **.zip myfile

unzip 解压缩文件

gzip **.** 压缩文件

tar -cvf **.tar myfile 归档文件，打包命令

tar -xvf **.tar 揭开归档

tar -cvzf **.tar.gz myfile 将归档后的文件进行gzip压缩

查找

locate keyword 此命令需要预先建立数据库，数据库默认每天更新一次，可用updatedb命令手工建立，更新数据库，快速查找

find 查找位置 查找参数 保证实时性 但是比较慢

find . -name *balabala* 通配符 0个或多个 ？表示一个

find / -name ***** 在整个根分区查找

find / -perm 777 指定权限查找(permission)

find / -type d

find . -name "a*" -exec ls -l {} \; 将找出的数据作为参数，传给 ls -l命令

-user -group -ctime(基于修改时间) -size(基于大小)
